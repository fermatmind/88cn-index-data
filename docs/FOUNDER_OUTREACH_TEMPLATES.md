# Founder Outreach Templates

Templates for reaching out to founders and maintainers of projects in the Seed 100. All outreach is optional, calm, and factual. Do not add promises, urgency, or speculation beyond what is written here.

---

## English

### X / Twitter DM

```
Hi <name>, we are building 88CN, a maintainer-reviewed AI project signal index.

We created a public structured profile draft for <project> based only on public sources. The goal is to make AI projects easier to understand through clean metadata, official source links and machine-readable public profiles.

If you are the founder or maintainer, you can claim or correct the profile here:
<claim-url>

This does not guarantee ranking, traffic or AI citations. It is simply a free structured public profile that you can review, update or request to remove.
```

### GitHub Issue / Discussion

```
Hi <name>,

We are building 88CN (<repo-url>), a maintainer-reviewed AI project signal index that creates structured public profiles for AI projects using only official public sources.

As part of this, we have created a draft profile for <project> based on the project's website, docs, GitHub, and other publicly available sources. You can find the profile at:

<profile-path>

If you are the founder or maintainer, you are welcome to:
- Review and correct any inaccuracies.
- Add or update official source links (docs, GitHub, pricing, changelog).
- Mark the profile as founder-claimed once it accurately reflects your project.
- Request removal if you prefer not to be listed.

To make changes, you can open a PR against this file, or let us know in this issue and a maintainer will handle it.

This is a free public resource. Inclusion does not guarantee ranking, traffic, or AI citations. No private data is collected or published.

Thank you.
```

### Product Hunt Comment

```
Hi <name>, we're building 88CN, a maintainer-reviewed AI project index. We created a structured public profile for <project> using only official public sources. If you'd like to review, correct, or claim it, the draft is here: <claim-url>. No promises on ranking or traffic — just a free public profile you can control. Congrats on the launch!
```

## What These Templates Must NOT Include

When customizing or translating these templates, the following must **never** appear:

- "backlink", "dofollow", "SEO juice"
- "guaranteed ranking", "guaranteed traffic", "guaranteed AI citation"
- "claim within X hours/days or else" or any urgency threat
- FOMO language ("don't miss out", "limited spots", "act now")
- Investment or funding promises
- Claims that inclusion improves search ranking
- Claims that AI systems (ChatGPT, Claude, etc.) will cite the profile
- Any language that pressures, manipulates, or creates anxiety
